# Recruiter Persona Mappings

## 1. Technical Hiring Manager (The Gatekeeper)
- **Priority**: Evidence of technical depth, methodology, and problem-solving.
- **Evidence Requirement**: Command logs, audit trails, and architecture diagrams.
- **Goal**: Verify the "How" and "Why" behind technical decisions.

## 2. Technical Recruiter (The Screener)
- **Priority**: Alignment with role requirements, keywords, and visual confirmation of results.
- **Evidence Requirement**: High-fidelity screenshots, clear impact summaries, and tech stack badges.
- **Goal**: Quickly match skills to the job description and verify "it works."

## 3. Product Lead (The Stakeholder)
- **Priority**: Impact on user experience, business value, and delivery speed.
- **Evidence Requirement**: Functional demo screenshots, user flow descriptions, and release readiness reports.
- **Goal**: Confirm that technical excellence translates to product quality.
